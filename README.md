EasyChord - ダイアトニックコード・リードシート作成ツール
音源を聴きながらステップ単位でコードを入力し、五線譜風のリードシートを素早く作成・PDF出力できるWEBアプリケーションです。

主な機能
・音源連動ステップ入力: ローカルの音源を再生しながら、数字キー（1〜7）でダイアトニックコードを直感的に連続入力できます。

・トランスポーズ機能: 読み込んだ音源のキーを調整しながらの作業が可能です。

・柔軟な楽譜構成: 楽曲の途中で小節ごとの「転調」や「拍子変更」を追加できます。

・シンコペーション対応: 拍と音符の種類（4分、8分、16分など）を指定して、リズムのキメ（音符）を視覚的に配置できます。

・カスタマイズ & PDF出力: タイトルやコードのフォント・文字サイズを変更し、そのまま五線譜プレビューからPDF出力が可能です。
